# BindDDLWithDatabase

BindDDLWithDatabase is a simple ASP.NET Core project that demonstrates how to bind a DropDownList (DDL) with data retrieved from a database using Entity Framework Core.

## Features
- Fetch data from a database
- Bind database records to a DropDownList
- Uses Entity Framework Core
- Beginner-friendly example

## Technologies Used
- ASP.NET Core
- Entity Framework Core
- SQL Server
- C#

## Project Purpose
This project is created for learning purposes to understand how database data can be displayed in a DropDownList in ASP.NET Core applications.
